# FUTURE_CS_02
# Phishing Email Detection & Awareness Report (Task 2)

## Objective
Analyze phishing email samples, identify phishing indicators, classify risk, and create an awareness guide for employees.

## Tools Used
- Google Message Header Analyzer: https://toolbox.googleapps.com/apps/messageheader/
- Browser link inspection (hover/copy links, no clicking)

## Analysis Approach
1. Collected phishing email samples (public datasets)
2. Reviewed email headers (SPF/DKIM/DMARC where available)
3. Inspected sender domains and links
4. Identified common phishing indicators
5. Classified each email as Safe / Phishing
6. Documented findings and prevention guidelines

- Screenshots/ (header analysis screenshots where applicable)
